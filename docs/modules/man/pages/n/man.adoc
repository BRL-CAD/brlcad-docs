== NAME

man - View manual pages.

== SYNOPSIS

*[cmd]#man#*  {[rep]_cmdname_}

== DESCRIPTION

Launches the *[cmd]#brlman#*  graphical man page viewer with the man page documenting the specified MGED command name.
