= Z(nged)

== NAME

Z - Clear the mged display.

== SYNOPSIS

*Z*

== DESCRIPTION

Zap (i.e., clear) the _mged_ display.

== EXAMPLES

Clear the _mged_ display. 

.Enter [cmd]*Z* to clear the _mged_ display. 
====
[prompt]#mged># [in]`Z`

The _mged_ display is clear. 
====
