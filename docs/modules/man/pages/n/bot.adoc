== NAME

bot - 
      Perform operations on a Bag of Triangles (BoT) primitive.
    

== SYNOPSIS

*bot* [subcommand] [options]
::
+
*get* {faces | minEdge | maxEdge | orientation | type | vertices} {input_bot}::
+
*chull* {input_bot} {output_bot}::
+
*solid* {input_bot}
== DESCRIPTION

The [cmd]*bot* command.
