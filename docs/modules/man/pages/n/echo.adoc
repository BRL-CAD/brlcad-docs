= ECHO(nged)

== NAME

echo - Echos whatever text is provided as an argument on the command
line. This is intended for use in MGED scripts.

== SYNOPSIS

*echo* {_text_} []

== DESCRIPTION

Echos whatever text is provided as an argument on the command
line. This is intended for use in _MGED_ scripts.

== EXAMPLES

The example shows the use of the [cmd]*echo* command in displaying
text.

.Using the echo command to display text.
====
[prompt]#mged># [in]`echo some text goes here`

Displays the text, "some text goes here".
====
