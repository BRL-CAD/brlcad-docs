= DPIX-PIX(1)

== NAME

dpix-pix - convert a double precision dpix file to a pix file

== SYNOPSIS

*dpix-pix* _file.dpix_ > _file.pix_

== DESCRIPTION

Outputs a dpix file to pix (see xref:man:5/pix.adoc[*pix*(5)])
format. Standard input is read if a file name is not supplied.
