== NAME

bot - 
      Perform operations on a Bag of Triangles (BoT) primitive.
    

== SYNOPSIS

*[cmd]#bot#*  [subcommand] [options]
::
+
*[cmd]#get#*  {faces | minEdge | maxEdge | orientation | type | vertices} {input_bot}::
+
*[cmd]#chull#*  {input_bot} {output_bot}::
+
*[cmd]#solid#*  {input_bot}
== DESCRIPTION

The *[cmd]#bot#*  command.
