== NAME

mrot - Rotates the view using model x y z.
   

== SYNOPSIS

*mrot* {_x y z_}

== DESCRIPTION

Rotates the view using model __x y z__.

The example shows the use of the [cmd]*mrot* command to rotate the view using model __x y z__.
