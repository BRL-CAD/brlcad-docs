== NAME

exit - Ends the MGED process.
   

== SYNOPSIS

*[cmd]#exit#* 

== DESCRIPTION

Ends the MGED process. It is a synonym for the *[cmd]#quit#* command. 

== EXAMPLES

The example shows the use of the *[cmd]#exit#*  command to stop MGED. 

.Using the *[cmd]#exit#*  command to end MGED.
====

[prompt]#mged># [ui]`exit` ::
MGED stops. 
====
