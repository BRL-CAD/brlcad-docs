== NAME

mrot - Rotates the view using model x y z.

== SYNOPSIS

*mrot* {_x y z_}

== DESCRIPTION

Rotates the view using model _x y z_. 

== EXAMPLES

The example shows the use of the [cmd]*mrot* command to rotate the
view using model _x y z_.
