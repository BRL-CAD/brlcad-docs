== NAME

refresh - Updates the MGED display.
   

== SYNOPSIS

*refresh*

== DESCRIPTION

Updates the MGED display. 

== EXAMPLES

The example shows the use of the [cmd]*refresh* command to update the MGED 	display. 

.Update the MGED display.
====
[prompt]#mged>#[ui]`refresh`

Updates the MGED display. 
====
