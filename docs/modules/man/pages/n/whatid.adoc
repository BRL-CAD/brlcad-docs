= WHATID(nged)

== NAME

whatid - Lists the ident number of the specified region.

== SYNOPSIS

*whatid* {_region_name_}

== DESCRIPTION

Lists the ident number of the specified _region_.

== EXAMPLES

The example shows the use of the [cmd]*whatid* command to list the
ident number of a specified region.

.List the ident number of a specified region.
====
[prompt]#mged># [ui]`whatid regiona`

Gets the ident number for regiona.
====
